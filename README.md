# Laundry Mart Login Page

A simple and responsive Login Page UI created using **HTML** and **CSS**. This project features a clean two-column layout with a laundry-themed illustration, login form, hover effects, and modern styling.

## Project Structure

```
Laundry-Mart/
│
├── index.html
├── style.css
└── README.md
```

## Features

* Responsive two-section layout
* Laundry-themed illustration
* Email and Password input fields
* Forgot Password link
* Login button with hover effect
* Register Now link
* Modern and clean user interface
* Flexbox-based layout

## Technologies Used

* HTML5
* CSS3

## How to Run

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Ensure `index.html` and `style.css` are in the same folder.
4. Open `index.html` in your browser.

### Using VS Code Live Server

1. Install the **Live Server** extension.
2. Right-click on `index.html`.
3. Select **Open with Live Server**.
4. The webpage will open automatically in your browser.
